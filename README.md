# lookStargram


### 요약
- 설명 : 2021년 국비지원 웹 개발자 양성과정에서 진행했던 프로젝트(팀)
- 기간 : 약 한달(2021.08.30 ~ 2021.09.24)
- 주제 : 자신의 체형에 맞는 다양한 스타일의 정보를 공유할 수 있는 패션 커뮤니티
- 사용기술
  - Backend: Java, MyBatis
  - FrontEnd: HTML/CSS, BootStrap, JavaScript
  - Tool : STS 4
  - DBMS: ORACLE
  - Server: Tomcat 8.5
  - 협업 도구: GitHub, slack

- 담당부분
 - 게시판 및 댓글 / DB 관리 / 형상관리(GitHub)
